# Proyecto Next.js + ShadCN + TweakCN

## 🛠️ Tecnologías

- ⚛️ **[Next.js](https://nextjs.org/)**
- 🎨 **[ShadCN/UI](https://ui.shadcn.dev/)**
- 🧩 **[TweakCN](https://tweakui.dev/)**
- 💨 **[Tailwind CSS](https://tailwindcss.com/)**
- 🧠 **TypeScript**

## ⚙️ Instalación

Cloná el repo y corré los siguientes comandos:

```bash
npm install
npm run dev
```
